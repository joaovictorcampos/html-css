Colocando cores por "nomes"

Método onde escrevemos o nome da cor para que ela seja utilizada no css, acaba sendo um método um pouco limitado pois dependemos das cores terem sido nomeadas para que possamos utiizá-las.

Colocando cores por "código Hexadecimal"

Método onde utilizamos códigos Hexadecimais para adicionar as cores que desejamos, é uma forma mais livre para adicioná-las pois utilizamos da "mistura" de cores nesse método, onde a sequência de seis digitos representam a quantidade de cada cor a ser utilizada.

Código= (# 00 - Red 00 - Green 00 - Blue) ----> Cores que fazem as misturas= (Red/Vermelho Green/Verde Blue/Azul) ----> Quantidades= (0 - 0= mínimo F - 255= máximo)

Colocando core por "Código RGB"

Método onde utilizamos o código RGB para adicionar as cores. Esse método funciona da mesma maneira que o código decimal, porém tem sua representação sendo como números comuns e não pela representação hexadecimal.

O código RGB vai do mínimo= 0 até o máximo= 255 de cada cor da escala RGB= R-RED, G-GREEN, B-BLUE.

Colocando cores por "Características das cores"

Método que utiliza as características das cores para adicioná-las, características estas que são: Matiz, Saturação e Luminosidade com a função hsl
